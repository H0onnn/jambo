# Jambo

## 프로젝트 소개

Jambo WebService

<br/>

## 프로젝트 실행 가이드

- 실행 방법 (2가지 중 택 1)
  > 1. ZIP 파일 다운로드 및 압축 풀기 후 코드 에디터로 실행
  > 2. 아래 커멘드를 이용한 실행

```bash
$ git clone https://github.com/H0onnn/jambo.git
$ npm install
$ npm run dev
```
